# LangBot: Telegram-бот для изучения английского языка

## Описание проекта
**LangBot** - это интерактивный помощник в мессенджере Telegram, разработанный в рамках дисциплины «Технологии разработки программных приложений». Бот помогает пользователям систематизировать изучение иностранных слов и закреплять знания грамматики.

**Основные возможности:**
* **Личный словарь:** добавление пар «слово - перевод» для индивидуального обучения.
* **Практика:** режим случайной проверки знаний по добавленному словарю.
* **Теория:** структурированные текстовые материалы по правилам английского языка (например, использование артиклей).

---

## Зависимости
Проект написан на языке **Python**. Для корректной работы требуются следующие библиотеки:

* `pyTelegramBotAPI` - основной фреймворк для работы с Telegram Bot API.
* `python-dotenv` - библиотека для безопасного хранения токена в переменных окружения.

> **Примечание:** Полный список зависимостей зафиксирован в файле `requirements.txt`.

---

## Установка и запуск

### 1. Подготовка окружения
Убедитесь, что у вас установлен Python версии 3.12 или выше. Склонируйте репозиторий и перейдите в папку проекта.

### 2. Установка библиотек
Выполните команду для установки необходимых пакетов:
```bash
pip install -r requirements.txt
```

### 3. Подготовка окружения
Создайте в корневой директории файл `.env` и добавьте туда ваш API-ключ, полученный от @BotFather:
```bash
TOKEN=ваш_телеграм_токен
```

### 4. Подготовка окружения
Запустите бота с помощью команды:
```bash
python main.py
```
